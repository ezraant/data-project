#How to run our program in the over/under mode

type: python nba_over_under.py
Follow the prompt to choose the over/under mode
Follow the prompt to enter the player's name
Follow the prompt to enter the opponent
Follow the prompt to enter the home/away status of the game
Follow the prompt to enter the desired statistics
Follow the prompt to enter the over/under lines

# How to run our program in the player statistic mode

type: python nba_over_under.py
Follow the prompt to enter the desired player name
Follow the prompt to choose if you want season averages or season averages against a specific opponent
Follow the prompt to enter either the year or the opponent, depending on what you chose above
Follow the prompt to choose either this season or career if you chose a specific opponent
